# EDA 2026-05-25

## Dataset Counts

| Split | Wells | Rows | Target Rows | GR Missing |
|---|---:|---:|---:|---:|
| train | 773 | 5,092,255 | 3,783,989 | 1,507,972 |
| test | 3 | 19,221 | 14,151 | 3,784 |

Target rows are rows where `TVT_input` is missing.

## Key Findings

- Train wells have a median of 6,576 rows.
- Train wells have a median of 4,840 target rows.
- Mean hidden fraction is 0.733; most wells require prediction for the majority of the lateral.
- Mean GR missing fraction is 0.294 and varies widely by well.
- `TVT_input` missing region is a contiguous tail, so Prediction Start can be detected reliably.
- Final target delta from anchor varies from about -104 ft to +90 ft.
- Test wells are present in train and must be treated as leakage for LB-only checks.

## Baselines

Naive all-train target-row scoring:

| Baseline | RMSE |
|---|---:|
| last known TVT anchor | 15.91 |
| naive slope over last 200 known rows | 123.76 |

Interpretation:

- Anchor is a strong and stable baseline.
- Naive slope extrapolation can explode, so slope features need clipping/robust fitting/model-based use rather than direct extrapolation.

## CV Implications

- Row-random CV is invalid because neighboring rows in the same well are highly correlated.
- Group by `well_id` is mandatory for real validation.
- Fold assignment should balance target-row counts, not only well counts.
- Evaluation should use only target rows where `TVT_input` is missing.
